# 경영성과관리 대시보드 시스템 (STMS)

## 1. 과제 배경

기업의 전략적 목표 달성을 위해 매년 수립되는 **중점추진과제**와 **OI 과제**를 체계적으로 관리하고, 각 부서의 월별 성과를 실시간으로 추적할 수 있는 통합 관리 시스템의 필요성이 대두되었습니다. 기존에는 엑셀 기반의 수동 관리로 인해 미입력 과제 파악의 어려움, 보고서 작성의 시간 소모, 성과 현황 파악의 지연 등의 문제점이 있었습니다.

## 2. 개요

**경영성과관리 대시보드 시스템(STMS)**은 전사 전략 과제의 등록, 추적, 성과 관리 및 AI 기반 보고서 생성을 지원하는 웹 기반 통합 관리 플랫폼입니다. 관리자는 모든 과제의 현황을 한눈에 파악하고, 담당자는 본인 과제의 활동내역을 효율적으로 입력할 수 있으며, AI가 월간/종합 보고서를 자동으로 생성합니다.

**개발 기간**: 2024년  
**개발 기술**: Spring Boot (Backend), React (Frontend)  
**주요 특징**: 실시간 현황 모니터링, AI 기반 자동 보고서 생성, 직관적인 대시보드 UI

## 3. AS-IS / TO-BE

### AS-IS (기존 방식)
- **과제 관리**: 엑셀 파일 기반 수동 관리
- **활동내역 입력**: 이메일 또는 공유 폴더 제출
- **현황 파악**: 월말 집계 후 수동 확인
- **보고서 작성**: 담당자가 직접 작성하여 시간 소모
- **문제점**: 미입력 과제 파악 어려움, 실시간 현황 파악 불가, 보고서 작성 부담

### TO-BE (개발 시스템)
- **과제 관리**: 웹 기반 통합 관리 시스템으로 등록/수정/조회 일원화
- **활동내역 입력**: 담당자별 화면에서 직접 입력, 이전 월 데이터 참고 가능
- **현황 파악**: 실시간 대시보드로 미입력 과제, 진행 상태, 달성률 즉시 확인
- **보고서 작성**: AI가 활동내역을 분석하여 월간/종합 보고서 자동 생성
- **개선 효과**: 실시간 모니터링, 자동화된 보고서 생성, 체계적인 성과 관리

## 4. 주요 기능

### 4.1 과제 관리
- **과제 등록/수정**: OI 과제 및 중점추진과제 등록, 카테고리 분류, 담당자 지정
- **성과 기준 설정**: 재무/비재무 분류, 정성/정량 평가, 건수/금액/% 지표 설정
- **진행 상태 관리**: 진행중/완료/지연/중단 상태 추적

### 4.2 활동내역 입력 및 성과 추적
- **월별 활동내역 입력**: 담당자가 매월 활동내역 직접 입력, 이전 월 데이터 참고
- **목표/실적/달성률 관리**: 월별 목표값, 실적값 입력, 달성률 자동 계산
- **파일 첨부**: 활동내역 관련 파일 업로드/다운로드 지원

### 4.3 통합 대시보드
- **관리자 대시보드**: 전체 과제 현황, 상태별 통계, 본부별 현황, 평균 달성률 조회
- **담당자 화면**: 본인 담당 과제만 조회, 이달 미입력 과제 우선 표시
- **필터링 및 정렬**: 상태, 평가기준, 담당 본부별 필터링, 다양한 정렬 옵션

### 4.4 AI 기반 보고서 생성
- **월간 보고서**: 이번 달 활동내역 기반 자동 생성
- **종합 보고서**: 전체 기간 활동내역 종합 분석
- **보고서 형식**: HTML(뉴스클립용), 마크다운(편집 가능) 형식 지원
- **프롬프트 수정**: 생성된 보고서를 추가 프롬프트로 수정 가능

### 4.5 알림 기능
- **활동 기일 요청**: 관리자가 담당자에게 활동내역 입력 기일 요청 알림 발송
- **일자 선택**: 알림 발송 일자 선택 가능, 다중 과제 일괄 발송 지원

## 5. 기대 효과

### 5.1 운영 효율성 향상
- **시간 절감**: AI 기반 자동 보고서 생성으로 보고서 작성 시간 80% 이상 절감
- **실시간 모니터링**: 월말 집계 없이 실시간으로 과제 현황 파악 가능
- **자동화**: 미입력 과제 자동 식별 및 알림 발송으로 관리 업무 효율화

### 5.2 의사결정 지원
- **데이터 기반 의사결정**: 실시간 성과 데이터를 통한 신속한 의사결정 지원
- **시각화**: 대시보드를 통한 직관적인 현황 파악
- **트렌드 분석**: 월별 달성률 추이를 통한 성과 트렌드 분석 가능

### 5.3 업무 품질 개선
- **체계적 관리**: 과제 등록부터 성과 추적까지 전 과정의 체계적 관리
- **일관성 확보**: 표준화된 입력 양식으로 데이터 일관성 확보
- **투명성 향상**: 실시간 현황 공유로 조직 내 투명성 향상

### 5.4 사용자 경험 개선
- **직관적 UI**: Apple 스타일의 심플하고 깔끔한 인터페이스
- **즉시성**: 화면 진입 시 이달 미입력 과제 즉시 식별
- **편의성**: 이전 월 데이터 참고, 파일 첨부 등 편의 기능 제공

**결론**: 본 시스템을 통해 전사 전략 과제의 체계적 관리와 실시간 성과 추적이 가능해지며, AI 기반 자동 보고서 생성으로 업무 효율성이 크게 향상될 것으로 기대됩니다.
